### Git 使用语法

git 常用命令：

​		git init		项目初始化（本地环境）

​			 clone	从远程克隆一个项目到本地

​			 status	查看当前库的状态

​			 reset	  将代码重置

​			 add		 将文件添加到暂存区

​			 commit	将修改提交到本地仓库

​			 push		 将本地库的内容推送到远程仓库

​			 pull			将远程仓库的更新内容拉取到本地

​			 branch	  分支管理

​			 checkout	切换分支  /  代码还原	/ 	代码回滚

​			 log				查看提交历史日志

​			 remote		远程管理

​			 diff				差异对比

​			 merge			合并
